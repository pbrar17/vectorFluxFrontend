About VectorFlux

VectorFlux is a web application designed to visualize vector fields defined by mathematical functions. Users can input their own functions for the x and y components of the vector field, and the application will render a graphical representation of the field.

How It Works

When you enter your functions for u(x,y) and v(x,y), the application sends these functions to a serverless backend that processes the request and generates an image of the vector field. The image is then displayed on the page.

Features

  • Input custom mathematical functions for vector fields
  • Real-time rendering of vector fields
  • Responsive design for various devices
